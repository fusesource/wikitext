Server Configuration


Although default settings are provided, the bugzilla server can be changed. To change the configuration, go to Preferences (in the Window menu) and open the Bugzilla tab.

Here, the following parameters can be set:

  1. URL of the Bugzilla server (set by default to the Eclipse Bugzilla server). This is used by the bug report viewer to view Bugzilla items within Eclipse.
  2. Your login name for the Bugzilla server (optional).
  3. Your password for the Bugzilla server (optional).
  4. Whether the version of the Bugzilla server to be accessed is 2.18 or 2.16 (Note: The Eclipse.org server is 2.18)

Your Bugzilla login name and password are only required if you wish to create or modify bugs using this Bugzilla plugin. Viewing bugs using this plugin does not require that a user name and password be specified unless it is required by the server.
