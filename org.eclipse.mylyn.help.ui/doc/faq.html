Frequent Asked Questions

What platforms are supported?

Mylar supports Eclipse 3.1 final and Eclipse 3.2 milestone releases.  For the time being Mylar only runs in a Java 5 VM, but we are exploring support for JDK 1.4 VMs.

How do I stop declarations from showing up in the Package Explorer?

If you don't like Mylar's constant showing of Java members in the Package Explorer, select the drop-down menu, then "Filters..." and enable the "Mylar Java Declarations Filter".  It will then stick in the menu in case you want to toggle between modes.

Why does nothing show up in the Active Search or Active Hierarchy?

As you work and elements become landmarks (bold decoration), and these elements populate the Active Search and Active Hierarchy views.  To force an element to populate the views manually make it a landmark by right-clicking or hitting Ctrl+Alt+Shift+Up Arrow.
