Mylar Tasks view tips

 
  • Can't find the task you're looking for? Start typing in the search box at the top of the view and matching tasks will be displayed in the tasks view.
    Tasks that have been work on in the past but are no longer present in the task view will be retrieved and displayed under the Archived Reports category.

 

Keyboard Shortcuts

 

View  Key Sequence Description
Mylar Tasks Create new task from browser URL Mozilla - drag URL from browser and drop on task list view.
Internet Explorer - drag URL from browser, hold down CTRL key and drop on task list view.
Package Explorer Ctrl-Left_Click Reveal elements hidden in view due to Mylar filtering
  Ctrl-Alt-Shift-Down Make the selected element less interesting. If the element is a landmark, this key sequence will remove the landmark.
