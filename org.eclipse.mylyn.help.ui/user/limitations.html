Known Limitations

Supported platforms

  • Eclipse 3.1 final or later (e.g. the 3.1m5a based JBossIDE is not supported)
  • Java 5 VM or later

Tasklist

Bugzilla refresh fails if task active on startup(Bug 104118)

Java editing

The Package Explorer can get into a mode where it refresh too often (Bug 103585).  The work-around is to Toggle the "Apply Mylar" glasses in the toolbar.

Folding must be toggled off in order to view all the matches for "mark occurrences".  This is not a Mylar-specific limitation but is more noticeable when Automatic Folding is enabled.

Task List

To select a task, the column with the icon or the space before the Mylar glasses has to be selected, since all other fields are editable.

Bugzilla

You can only set one bug repository per workspace.  Proxy servers are not yet supported.
