Mylar Overview

This initial release of Mylar is not well documented.  More complete documentation in the 0.3.2 release, scheduled for July 18, 2005.  Please check the update site again on Tuesday July 12 for a an interim update.  For questions about usage please email mylar-users@eclipse.org or mylar@cs.ubc.ca

To start using Mylar

  • Open the Mylar / Task List view
  • Create a new task
    • Tasks can also be bug reports, but you must configure the Bugzilla server first (Preferences -> Bugzilla)
  • Activate a task by clicking the gray outline of the glasses to the left of the task icon.  Deactivate by clicking again.
  • Start working on the task, Mylar will be monitoring the task context
    • After you deactivate the task the context will be saved for re-activating at a later time
  • Whenever the Mylar glasses appear on the toolbar of a view (e.g. Outline, Package Explorer, Problems List) you can click these in order to enable Mylar degree-of-interest filtering.  Toggle this button to return the view to its original state.
    • To temporarily show all of the children for a node, <alt>click the node.  Note that this won't work for the currently-selected view.
  • In addition to filtering, the Task Context can also be highlighted.  Right click a task to select a highlighter.
    • Note that to select an item you have to click the space before the glasses or the icon.
